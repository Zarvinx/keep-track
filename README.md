# Keep Track

> **Note**: This is a fork of [red-coracle/episodes](https://github.com/red-coracle/episodes) with the UI rewritten in Kotlin and Jetpack Compose.

Keep track of which episodes you've watched of your favourite shows.


This product uses the [TMDB](https://www.themoviedb.org) API but is not endorsed or certified by TMDB.

## Contributing

Contributions are very welcome. Please file bugs, fork the repository, [translate](https://hosted.weblate.org/projects/episodes/) and send pull requests.

## Dependency and Security Audit

Run dependency freshness and vulnerability checks with:

```bash
./gradlew dependencyAudit
```

This runs:
- `:app:dependencyUpdates` (outdated dependency report)
- `dependencyCheckAnalyze` (known CVE scan)
